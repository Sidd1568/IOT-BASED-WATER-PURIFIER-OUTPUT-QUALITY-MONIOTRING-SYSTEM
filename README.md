# Water Quality Monitor using ESP32

This project monitors water quality using a TDS sensor and displays the value on an OLED display.

## Components
- ESP32
- TDS Sensor
- OLED Display (SSD1306)
- Active Buzzer

## Features
- Real-time TDS monitoring
- OLED display output
- Buzzer alert for poor water quality

## Technologies
- Arduino (C++)
- Adafruit SSD1306 Library
- Adafruit GFX Library
